# 📞 ChurnGuard — Прогноз оттока клиентов телеком-компании

ML-система для предсказания вероятности ухода клиента (churn) на основе 19 признаков.  
Разработано как pet-проект для демонстрации полного пайплайна: EDA → моделирование → интерпретация → веб-интерфейс.

---

## Результаты

| Модель               | Accuracy | Precision | Recall | F1-Score | ROC-AUC |
|----------------------|----------|-----------|--------|----------|---------|
| **XGBoost (tuned)**  | 0.794    | 0.638     | 0.519  | 0.572    | **0.838** |
| Logistic Regression  | 0.794    | 0.624     | 0.564  | **0.593**| 0.835   |

> 🔍 Ключевой инсайт: **тип контракта** (`Contract`) — самый важный признак. Клиенты с *месячным контрактом* имеют наибольший риск ухода.

## Как запустить

1. Убедитесь, что у вас установлен Python 3.9+
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
3. Убедитесь, что в папке models/ находятся обученные модели:
- best_xgb_model.pkl
- scaler.pkl 
4. Запустите Streamlit-приложение:
   ```bash
   streamlit run app.py
